# Android (Kotlin) companion project

The mobile rebuild lives in a **sibling folder** (not inside this repo):

```
D:\Projects\
  dune-tracker\           ← this Django app (reference)
  dune-tracker-android\   ← Kotlin learning project
```

**Start here:** [../dune-tracker-android/README.md](../dune-tracker-android/README.md)

**Learning curriculum:** [../dune-tracker-android/docs/LEARNING_PATH.md](../dune-tracker-android/docs/LEARNING_PATH.md)

The Android app targets **offline-first** SQLite on device. Feature parity and scoring rules are documented against `games/models.py`, `games/scoring.py`, and templates in this repo.
